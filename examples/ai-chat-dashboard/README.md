# Example: AI Chat Dashboard (stub)

Same idea as `crm-starter/` — a starting prompt, not a pre-filled answer.

## Idea

A chat widget backed by a real LLM API, plus a simple dashboard showing message volume and a list of past conversations.

## Run it

```
Build an MVP: a chat widget backed by a real LLM API, with a simple dashboard
showing message volume and a list of past conversations.

Use this repository. Follow the full workflow (research → prd → roadmap → ui →
build → debug → deploy) and save the output of each stage into this folder,
the same way examples/booking-system/ is structured.
```

Note: this one needs an actual LLM API key to get past `/backend` — decide which provider before starting, and treat "no API key available right now" as a real constraint for `/research` and `/prd` to account for, not something to quietly work around.

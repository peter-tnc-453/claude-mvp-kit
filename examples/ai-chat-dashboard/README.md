# Example: AI Chat Dashboard (stub)

Same idea as `crm-starter/` — a starting prompt, not a pre-filled answer.

## Idea
A chat widget backed by an LLM API, plus a simple dashboard showing message volume and a list of past conversations.

## Run it

```
สร้าง MVP: แชตบอทที่ต่อ LLM API จริง พร้อม dashboard ง่ายๆ ที่โชว์จำนวนข้อความ
และ list การสนทนาที่ผ่านมา

ใช้ repository นี้ ทำตาม workflow เต็ม (research → prd → roadmap → ui → build →
debug → deploy) เก็บผลลัพธ์แต่ละขั้นตอนไว้ในโฟลเดอร์นี้เหมือน examples/booking-system/
```

Note: this one needs an actual LLM API key to run past `/backend` — decide which provider before starting, and treat "no API key available right now" as a real constraint `/research` and `/prd` should account for, not something to work around silently.
